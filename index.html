PhD Researcher and Data Scientist at Pacific Northwest National Laboratory

I am researcher, data scientist, and engineer. My research interest include topological data analysis, dynamical systems theory, complex networks, time series analysis, computer vision, hypergraphs and set theory.

Research Interests

Complex Networks

Graph (network) representations of time series data by partitioning the state space reconstruction.

Dynamical Systems

Methods for studying bifurcations in dynamical systems.

Topological Data Analysis

Studying the shape of data through persistent homology.

Time Series Analysis

Time series analysis tools for studying the dynamic state of the underlying systems.

Computer Vision

Synthetic imagery for improving object detection performance and generative adversarial networks for image augmentation.

Hypergraphs

Studying hypergraph representations of data with an emphasis on temporal hypergraphs and studying that changing shape using zigzag persistence.
